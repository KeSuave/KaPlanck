# KaPlanck

A KAPLAY plugin that integrates [PlanckJS](https://piqnt.com/planck.js), while keeping the simple/fun API of [KAPLAY](https://kaplayjs.com/).

## PlanckJS

Planck.js is JavaScript/TypeScript rewrite of the [Box2D](https://box2d.org/) C++ physics engine for cross-platform game development.

## KAPLAY

KAPLAY (/ˈkæpleɪ/) is a JavaScript and TypeScript game library that makes it easy to create games for the web. It's free, open-source and fun. 

### Before You Start

Please read both [PlanckJS'](https://piqnt.com/planck.js) and [KAPLAY's](https://kaplayjs.com/) documentation. Understanding how each of the libraries
work is crucial to effectively use KaPlanck. Once you have a good grasp on both libraries, you can proceed to [Getting Started](/getting-started).
